# css的像素概念

css像素是个抽象的像素单位
设备像素是显示器的物理像素，是定死的数据
设备独立像素是物理像素*屏幕缩放的像素数
iPhone 3gs 物理像素（320* 480）
iPhone 4 物理像素（640*960）
但两者屏幕大小一致，所以在4上一个设备独立像素会对应两个设备像素进行渲染，而3上一个物理像素对应一个设备像素

设备像素 / 设备独立像素  的比值为 DPR
